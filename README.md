# dotfiles

This is based on Zach Holman's great "@holman does dotfiles".
You might want to [read the original post on the subject by Zach
Holman](http://zachholman.com/2010/08/dotfiles-are-meant-to-be-forked/).

To install, run this:

```sh
git clone https://github.com/mixmaster/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
git submodule init
git submodule update
script/bootstrap
```
